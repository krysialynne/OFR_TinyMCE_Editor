**PCF TinyMCE Rich Text Editor (GCC-Compliant)**

A Custom PowerApps Component Framework (PCF) Rich Text Editor based on TinyMCE, tailored specifically for Microsoft GCC (Government Community Cloud) environments where no external network calls are allowed.

The control is fully self-contained, bundling all required scripts, skins, fonts, and assets locally to ensure 100% offline operation and strict compliance with GCC security constraints.

Key capabilities include:

End-to-end HTML lifecycle support (read → edit → save) via content_html binding

Zero external HTTP requests during load and runtime

Sticky toolbar with smooth behavior during heavy content editing

Configurable font dropdown with controlled defaults

Clean, restricted UI (e.g., removed File menu) to meet enterprise UX standards

Delivered as a managed solution with verified metadata and seamless import into PowerApps

Optimized for stability, performance, and maintainability in regulated environments

Built to be easily embedded in PowerApps screens, with simple resizing and configuration after placement.
